# Stockholm Cyber Security Job Agent

AI-powered job scanning agent focused on **Cyber Security, Security Architect,
and CISO roles in Stockholm only**.

## Features
- Stockholm-only filtering
- AI-based relevance classification
- AI-based job scoring
- Deduplication via SQLite
- Email notifications
- Fully automated (cron-friendly)

## Requirements
- Python 3.10+
- OpenAI API key
- Email SMTP account

## Installation

```bash
git clone https://github.com/YOURNAME/stockholm-cyber-job-agent.git
cd stockholm-cyber-job-agent
